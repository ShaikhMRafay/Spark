# Spark

**Minimum iOS Version:** 16.2  
**Xcode Version:** 14.2
**Built With:** Swift-UIKit
**Dependency Manager:** CocoaPods  
**Pods Used:**  
- `SkeletonView`

---

## Project Overview
Spark is a professional iOS application developed using **MVVM clean architecture**. The app focuses on efficient data handling, UI responsiveness, and reusable components, providing a smooth experience for end-users.

---

## Features

### Networking
- Generic network handler for API requests  
- Fetch **rosters** and **contacts** dynamically  
- Caching mechanism using local JSON files for faster data access  

### UI Components
- **TableView** with custom XIB cells  
- **CollectionView** with custom XIB cells  
- Custom image loader for optimized image rendering  
- Custom toast messages for popups  
- Search feature:
  - Search by contact name  
  - Search by class roster name  
  - Search by contact ID  

### Utilities
- utilities for consistent date display  
- Reusable components for scalability and maintainability  

---

## Architecture
- MVVM (Model-View-ViewModel) clean architecture  
- Emphasis on modular, reusable, and testable code  

---

## Requirements
- iOS 16.2 or higher  
- Xcode 14.2 or higher  
- CocoaPods installed
